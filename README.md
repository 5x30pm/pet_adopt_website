# 🐾 Pet Adoption Website

## 📌 Description

This is a simple pet adoption website built using HTML and CSS.
It allows users to browse pets available for adoption and understand the adoption process.

---

## 🌐 Live Demo

👉 https://5x30pm.github.io/pet_adopt_website/

---

## 🚀 Features

* 🐶 Browse pets available for adoption
* 📋 Simple adoption process section
* 🎨 Clean and user-friendly UI
* 📱 Responsive design for mobile and desktop

---

## 🛠️ Technologies Used

* HTML5
* CSS

---

## ▶️ How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/5x30pm/pet_adopt_website.git
   ```
2. Open the project folder
3. Open `index.html` in your browser
